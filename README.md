# 下載網站小說
> 此程式適用下載完結小說，下載的章節內容會統一存在同個txt檔

> 程式碼若有遇到問題，建議至[ps_code](https://github.com/JhihHan/download_novel/blob/main/ps_code.md)查看，也歡迎跟我提出

### 實際操作
> 1.至小說網站選擇要下載的小說
